Start

To start an HTML project you will need a folder called root folder. This is the folder where all of your website files will be stored. The main page file for your website is always called intex.html. You can download already prepared root folder for you to start clean.

<link>

  1. Fonts

    If you want to change your website's font you will need to put <link> in the head tag of your site. Put href="" and rel="stylesheet" in the <link> tag. Add a link to the font library in the href's "".

    Example:

    <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

    The best place for you to fined high quality free fonts is Google Fonts website

  2. SCC Linking

    If you want to link a CSS file to your html file you will need to put <link> in the head tag of your site. Put type="text/css", rel="stylesheet" and href="" in the <link> tag. Add a relative path to the CSS file in the href's "".

    Example:

    <link type="text/css" rel="stylesheet" href="Files/Style.css">

  3. Tab icon

    If you want to put a custom tab icon on your website you will need to put <link> in the head tag of your site. Put type="image", rel="icon", sizes = "16x16" and href="" in the <link> tag. Add a relative path to the image you want in the href's "".

    Example:

    <link type="image" rel="icon" sizes="16x16" href="Images/tabLogo.png">

    Keep in mind that your tab logo does not need to to be 16x16 pixels. It just needs to be 1:1 ratio or it will not show on the tab.

<meta>

Meta tags are used to give information for the website without actually displaying it on the viewport.

  1. Adding a description for your website

    The text that you see under the website's title when you share a link is the text that you should include in this tag. You will need to put <meta> in the head tag of your site. Put name="description" and content="" in the <meta> tag. Add the text you want in the content's "".

    Example:

    <meta name="description" content="This is cool website. Come check it out!">

    Keep in mind that generally you do not have a limit in the words but Google generally snips the text to 155/160 characters. I recommend you use maximum of 100 characters as this is the ultimate.

  2. Adding a keywords for your website

    All search engines use keywords to find the right links to show the user. More keywords means better search rates. You will need to put <meta> in the head tag of your site. Put name="keywords" and content="" in the <meta> tag. Add all the keywords you want in the content's "" using , to separate them from each other.

    Example:

    <meta name="keywords" content="Example, Cold, Milk">

    There is no official limit on the number of keywords you can have. Each search engine sets their own rules so feel free to put as many keywords as you think are necessary but do not exaggerate.

  3. Website's author

    If ypu want to asigne your website you will need to put <meta> in the head tag of your site. Put name="author" and content="" in the <meta> tag. Add Your name or unername in the content's "".

    Example:

    <meta name="author" content="Jim Oxford">

  4. Viewport settings

    This is one of the most necessary tags your website needs. Put the line shown in the example box below in your head tag.

    Example:

    <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1">

    This tag scales the website in the same ratio the device is. This way everything on the page stays in the same order and scale as the device's screen.
